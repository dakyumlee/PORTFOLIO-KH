# KH Portfolio

이기현의 영화 포트폴리오 웹사이트

## Features

- 원페이지 시네마틱 포트폴리오
- 필모그래피 관리
- 갤러리 (Cloudinary 이미지 업로드)
- YouTube 영상 자동 임베드
- 관리자 대시보드
- PWA 지원
- 모바일 반응형 (햄버거 메뉴)

## Tech Stack

- Spring Boot 3.2.5
- Thymeleaf
- PostgreSQL
- Cloudinary
- Docker

## Design

- 다크 테마 시네마틱 UI
- 컬러 팔레트: #0D0D0D, #404040, #737373, #BFBFBF, #D9D9D9
- 폰트: Pretendard
